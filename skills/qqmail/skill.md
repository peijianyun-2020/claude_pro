# QQ邮箱读取工具

读取QQ邮箱的最近N封邮件内容。

## 使用场景

当用户想要：
- 读取QQ邮箱的最新邮件
- 查看最近的邮件列表
- 获取特定邮件的详细内容
- 监控邮箱的新邮件

## 配置说明

此SKILL使用以下环境变量（已配置在项目根目录的.env文件中）：
- `QQMAIL_EMAIL`: 你的QQ邮箱地址
- `QQMAIL_AUTH_CODE`: QQ邮箱授权码

## 使用方法

在Claude Code对话中使用：

```
/qqmail           # 读取最近10封邮件
/qqmail 20        # 读取最近20封邮件
```

或者直接告诉Claude：
- "帮我查看邮箱"
- "读取最新的邮件"
- "看看最近有什么邮件"

## 参数说明

- `count`: 读取邮件数量，默认10封

## 注意事项

1. 授权码等同于邮箱密码，已安全存储在.env文件中
2. QQ邮箱对IMAP连接有频率限制，避免频繁调用
3. 如遇到"登录失败"错误，可能需要重新生成授权码

## 技术实现

- 协议: IMAP (imap.qq.com:993 SSL)
- 脚本位置: `.claude/skills/qqmail_reader.py`
- 配置文件: `.env`
